#文档级邀请

#邀请机制

mdocs 支持文档级别的邀请，无需用户预先注册账户。

#生成邀请链接

在文档页面点击「邀请」按钮，生成一个带有时效和权限的分享链接。

#权限控制

  • 只读：可查看，不可编辑
  • 评论：可查看并添加评论
  • 编辑：可编辑文档内容

#安全性

邀请链接包含加密 token，支持设置有效期和访问次数限制。
