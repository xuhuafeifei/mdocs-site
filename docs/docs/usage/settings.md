# 设置页面

## 功能概述

设置页面是 mdocs 的集中配置中心，提供访客身份、域管理、内容管理等各种系统级功能。

## 入口

点击左侧边栏**顶部的 ⚙ 设置按钮**（在域名选择下拉左侧），进入设置页面。

---

## 左侧导航概览

设置页面左侧分为以下功能模块：

| 导航项 | 功能 |
|-------|------|
| [通用](#通用设置) | 语言切换、自动编辑、CLI Token、恢复码 |
| [我的收藏](./bookmarks.md) | 收藏的文档列表，支持搜索、取消收藏 |
| [我的文章](./my-documents.md) | 自己创建的所有文档，支持邀请成员 |
| [域管理](../usage/domain-members.md#域管理) | 创建、重命名、删除域，配置域权限与成员 |
| 成员模板 | 保存常用访客名单，供受限域批量邀请复用 |
| [保存与发布](./drafts.md) | 自动同步开关、未发布草稿列表 |

---

## 通用设置

### 语言切换

支持中英文双语界面：

- **中文**：完整本地化的操作提示与说明
- **English**：英文界面适配国际团队

点击「中」/「EN」按钮即时切换，无需刷新页面。

### 自动编辑开关

开启后，打开文档时自动进入编辑模式；关闭时默认以只读模式打开，需要手动点击编辑按钮。

适合不同的使用习惯：
- **写作为主**：建议开启，减少一次点击
- **阅读为主**：建议关闭，避免误触编辑

### CLI Token

用于命令行工具和自动化脚本的身份凭证。详情请参考 [CLI Token](./cli-token.md)。

### 恢复码

用于访客身份恢复。详情请参考 [恢复码](./recovery-code.md)。

---

## 保存与发布

### 自动同步开关

开启后，编辑停止 30 秒后自动将草稿同步到服务器。建议保持开启以获得最佳协作体验。

详情请参考 [草稿与同步](./drafts.md)。

### 未发布草稿

显示当前所有存在本地草稿但尚未发布到服务器的文档。点击可直接进入对应文档继续编辑或发布。

---

## 通用操作

### 返回文档

设置页面的任何位置，都可以通过以下方式回到文档视图：

1. 点击左侧导航底部的 **返回文档** 按钮
2. 点击浏览器后退按钮

## 常见问题

### 设置会保存在哪里？

大部分设置（如语言、自动编辑、自动同步）保存在浏览器 localStorage，只对当前浏览器生效。

以下数据保存在服务器，与访客身份绑定：
- 收藏列表
- 域与成员配置
- 成员模板
- CLI Token
- 恢复码

### 为什么设置页面看不到某些功能？

部分功能有前置条件：
- **域管理**：需要是域的创建者才能看到重命名、删除等操作
- **成员模板**：至少创建过一个模板后才有内容
- **邀请成员**：只有自己创建的文档才显示该按钮

### 设置页面支持键盘操作吗？

设置页面的输入框和按钮均支持 Tab 切换和 Enter 确认。表格列标题支持点击排序。
