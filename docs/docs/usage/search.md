# 全文搜索

mdocs 内置了基于 FTS5（SQLite 全文索引引擎）的搜索功能，支持中文分词。

## 搜索入口

在编辑器顶部的搜索栏输入关键词，系统会在当前域的所有文档中检索。

## 搜索能力

- **全文检索**：搜索文档正文内容，不只是标题
- **中文分词**：使用 jieba 分词，自动切分中文句子，提高匹配精度
- **相关性排序**：按 BM25 算法排序，匹配度最高的结果排在最前
- **权限过滤**：自动过滤当前访客无权阅读的文档
- **片段高亮**：搜索结果展示匹配片段，快速定位关键内容

## 限制搜索范围

支持通过 `--domain` 参数限定在特定域内搜索（仅 CLI 客户端支持），Web 端默认在当前域内搜索。

## 索引更新

- 文档创建或更新后自动触发索引重建（异步）
- 每 60 秒扫描一次待索引的文档
- 索引更新不影响文档的正常访问
