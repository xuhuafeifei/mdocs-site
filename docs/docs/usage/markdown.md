# Markdown 编辑

## 完整语法支持

mdocs 支持标准 Markdown 语法，包括：

- 标题（H1-H6）
- 加粗、斜体、删除线
- 链接、图片
- 列表（有序/无序）
- 表格
- 代码块（语法高亮）
- 数学公式（KaTeX）

## 富文本工具栏

编辑时提供所见即所得的工具栏：

- 快捷插入标题、列表
- 一键插入链接和图片
- 表格可视化编辑

## 流程图

使用 Mermaid 语法直接绘制流程图：

```mermaid
graph LR
    A[开始] --> B[编辑]
    B --> C[保存]
    C --> D[同步]
```
