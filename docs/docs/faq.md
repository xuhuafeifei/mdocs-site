# FAQ

## 常见问题

### mdocs 和 Notion/Obsidian 有什么区别？

mdocs 是纯私有化部署的开源知识库，数据完全在你自己的服务器上。

### 需要安装数据库吗？

不需要。mdocs 使用 SQLite，数据存储在你指定的目录中，无需额外安装任何数据库。

### 支持多人协作吗？

支持。通过文档级邀请链接，无需注册账户即可协作编辑。

### 数据如何备份？

直接复制整个 mdocs 目录即可完成备份。所有文档以 Markdown 格式存储，元数据在 SQLite 文件中。

### 支持哪些 Markdown 扩展？

完整支持标准 Markdown 语法，以及 Mermaid 图表、KaTeX 数学公式、代码高亮等。

### 如何升级？

下载新版本二进制文件替换即可。数据库和文档格式向后兼容。
