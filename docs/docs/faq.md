# FAQ

### mdocs 和 Notion / Obsidian 有什么区别？

**Notion** 是 SaaS 服务，数据在云端，隐私受制于服务商；**Obsidian** 是本地优先，但多人协作需要额外付费方案。mdocs 在这两者之间取了一个平衡：**纯私有化部署 + 无账户协作**。你控制服务器，但协作者不需要注册任何账户。

### 需要安装数据库吗？

不需要。mdocs 使用 SQLite，数据存储在 `~/.mdocs/` 目录中，无需单独安装或管理任何数据库服务。

### 数据存在哪里？

- **文档内容**：以 Lexical JSON 格式存储在文件系统
- **元数据**（索引、权限、访客信息）：存储在 SQLite 数据库中
- **未发布的草稿**：存储在浏览器 IndexedDB 中
- 备份时只需复制整个数据目录即可

### 支持多人协作吗？

支持。通过「域 + 文档级邀请」实现：
- public 域的文档默认可被所有人查看
- restricted 域需要是域成员才能看到文档
- 文档所有者可以邀请特定访客获得编辑权限
- 不需要协作者注册账户

### 适合多大规模？

mdocs 主要面向**小团队**（几人到十几人）。没有用户系统、没有复杂的权限层级，这是有意为之的简化。

### 数据如何备份？

直接复制 `~/.mdocs/` 目录即可。文档存储格式为 JSON，需要搭配 SQLite 元数据才能完整恢复。

### 能不能导出数据？

目前文档以 Lexical JSON 格式存储，只能由 mdocs 加载。未来会提供 Markdown 导出功能。元数据在 SQLite 中，可以用 SQLite 工具直接读取。

### 如何升级？

拉取最新代码 → `pnpm install && pnpm build` → 重启进程。数据库和文档格式保持向后兼容。
